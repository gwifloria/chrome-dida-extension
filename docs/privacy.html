English Version

Overview

"First Glance" is a Chrome browser extension that displays Dida365 (TickTick) tasks on your new tab page. This privacy policy explains how we collect, use, and protect your information.

Data Collection

This extension collects and stores the following data:

  • Dida365 Account Authorization
    • OAuth access tokens (to access your Dida365 data)
    • Tokens are stored in browser local storage
  • User Preferences
    • Theme selection
    • Default project settings
    • Stored in Chrome sync storage
  • Task Data Cache
    • Tasks and projects fetched from Dida365 API
    • Cached locally only, to improve loading speed

Data Usage

  • All data is used solely for the extension's functionality
  • We do NOT send your data to any third-party servers
  • We do NOT collect any analytics or tracking data

Third-Party Services

This extension uses the official Dida365/TickTick API:

  • API endpoint: api.dida365.com
  • Your data is protected by Dida365's privacy policy
  • See: https://dida365.com/about/privacy

Data Storage Location

  • All data is stored locally in your browser
  • Uses Chrome Storage API (supports cross-device sync for preferences)
  • We have no servers and store no user data

Data Deletion

You can at any time:

  • Log out in the extension to clear authorization
  • Uninstall the extension to delete all local data
  • Revoke app access in your Dida365 account settings

Permissions Explained

This extension requests the following permissions:

  • storage: Store user preferences and cache
  • identity: OAuth login flow
  • Access to api.dida365.com: Fetch task data

Contact

For privacy-related questions, please contact us via:

  • GitHub Issues: https://github.com/gwifloria/chrome-dida-extension/issues
